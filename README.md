* {ODPOWIEDŹ 3}
* {ODPOWIEDŹ 4}
{Pusta linia}
> {KOMENTARZE|UWAGI|LINKI|WYJAŚNIENIA|itp}  
> {KOMENTARZE|UWAGI|LINKI|WYJAŚNIENIA|itp}
